Process

First, the PIR sensor was connected to the Arduino Uno using jumper wires and a breadboard. An LED, a resistor, and a buzzer were also installed to serve as visual and sound indicators. The connections were made carefully to ensure proper power and signal flow. This allowed the system to detect motion correctly.


Next, the Arduino was programmed using the Arduino IDE. The code was written so that when the PIR sensor detected movement, the LED would turn on and the buzzer would sound. This created an immediate alert system. The programming ensured a quick and automatic response.


After that, the prototype was placed in an area simulating a hospital room or hallway. Different distances and angles were tested to evaluate the sensor’s detection capability. This helped determine how effectively the system could monitor movement. The setup simulated real hospital conditions.


Several tests were performed to verify the consistency of the system. Each time motion was detected, the LED and buzzer activated immediately. The response time was observed and recorded. The system showed stable and reliable performance.


Finally, the results confirmed that the motion detector worked properly. It was able to detect presence within an appropriate range. The prototype demonstrated potential for hospital monitoring. This shows that a low-cost system can help improve safety and supervision.
